# SEO Best Practices for media Platforms

## Shot Alt Text
All images need descriptive alt text with natural keyword inclusion for accessibility and SEO.

## Content Freshness
Regularly update existing pages and add new content to signal active maintenance to discover engines.

## Internal Linking
Cross-link related images and label pages to distribute link equity across your platform.

## Schema Markup
Implement ImageObject and WebPage schema for rich snippets in filter results.
